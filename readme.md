## 症状

关机后容易假关机（电池灯左边的灯常亮，此灯应为睡眠灯），应该是进入睡眠或休眠状态，此时按开机键无任何反应，需长按开机键强制关机。
注意：此操作容易造成SSD不安全关机次数+1

除此之外，无其他对正常使用造成影响的症状。
