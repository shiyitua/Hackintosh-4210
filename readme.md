# 补充说明

## 机型

ASUS R556L（华硕 R556L）

## 配置

| 配件   | 详情                                |
| :----- | :---------------------------------- |
| CPU    | Core i5-4210u                       |
| iGPU   | HD 4400                             |
| dGPU   | Nvidia GeForce 820m (2G)            |
| Memory | 12GB ddr3                           |
| SSD    | Move speed 960GB + Move speed 512GB |
| NIC    | Intel AC 8265                       |

声卡厂商是ALC的，具体型号忘记

## 症状

关机后容易假关机（电池灯左边的灯常亮，此灯应为睡眠灯），应该是进入睡眠或休眠状态，此时按开机键无任何反应，需长按开机键强制关机。
注意：此操作容易造成SSD不安全关机次数+1

除此之外，无其他对正常使用造成影响的症状。

## 备注

**隔壁仓库`Hackintosh-4460`的EFI文件**就是由本机型的EFI文件小改而来，只更改了`config.plist`文件，驱动和其他文件未做改动

- **截至目前2026.03.25未做改动，不保证后期对驱动文件进行更新，因为该EFI文件在开机时有一定概率会造成4460主机死机**
- **注意：上述主语为4460台式机，而非本机4210的笔记本**
